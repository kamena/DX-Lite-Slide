== Description ==
Lite and simple slider for your WordPress project. Works with placing a shortcode.

== Installation ==
1. Upload `DX-Lite-Slide` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Use `[dx_display_slideshow]` shortcode in your Post/Page or place `<?php echo do_shortcode( '[dx_display_slideshow]' ) ?>` in your templates.

== Frequently Asked Questions ==
= How to use plugin? =

Create few Slides with featured images and add `[dx_display_slideshow]` shortcode.
